# Kalavidara-Balaga

## Overview
Kalavidara-Balaga is an Android application developed to connect traditional folk artist troupes with event organizers and urban audiences. The platform helps artists showcase their performances, receive booking opportunities, and promote Karnataka’s cultural heritage digitally.

---

## Problem Statement
Traditional folk artists often receive work only during festivals and wedding seasons. Due to limited exposure and lack of digital presence, they struggle to connect with event planners throughout the year.

This application aims to bridge the gap between rural artists and urban event organizers.

---

## Features
- Artist Profile Management
- Search by District and Art Type
- Performance Gallery
- Direct Call Integration
- Equipment and Service Area Details
- Responsive UI/UX
- Firebase Database Integration

## Modules

### Home Screen
Displays featured artist troupes and categories.

### Artist Directory
Allows users to search and filter artists.

### Artist Profile
Displays troupe information, gallery, and contact details.

### Gallery
Shows performance images and media content.

### Booking Inquiry
Enables direct communication using call functionality.
